# 审美哲学 · 把 HTML 拉到"高级感"的 4 个维度

> 这是 CCBGZZY 设计的灵魂。前面的 token/字号/表格是"骨"，这一篇是"气"。
> 做任何页面前先读这一篇，定调子，再动手。核心一句话：**高级 = 低饱和同色系 + 卡片分层 + 气质匹配 + 可亲近的克制。**

## 0. 硬性版式规矩（不可违背）

1. **一屏一个核心观点。** 每屏/每个区块只讲一件事，把它讲到位；不要一屏塞五个重点。
2. **关键词只换 1–2 处色。** 一屏里用 `--accent` 点亮的关键词/数字最多 1–2 个，其余靠加重/留白。
3. **圆角 14–16px。** 卡片/按钮统一 `--radius`(14) / `--radius-lg`(16)，不要忽大忽小。
4. **留白 ≥ 20%。** 页面至少两成是空的——宁可空，不要挤。padding 用 `--space-5/6/7` 级别。
5. **用 `#1A1A1A`，不用 `#000`。** 文字/强调黑一律 `--title`(=#1A1A1A)；纯黑太硬，禁用。

## 维度一 · 颜色：低饱和 + 同色系（莫兰迪）

不是"调好看一点"，是把饱和度压下去、锁住一个色相。

- **莫兰迪色**：所有颜色都掺一点灰——雾蓝、苔绿、砖红那种"安静色"，不要糖果色、不要荧光。
- **水彩晕染感（watercolor wash）**：饱和度再压低一档，色块之间柔和过渡（注意：用纯色台阶实现层次，**不用渐变**——见配色铁律第 8 条）。
- **单色相锁定（monochromatic）**：一个页面锁一个主色相，层次靠 `--seq-1..5` 明度台阶拉开。AI 想乱也乱不起来。
- 强调只留一处暖色 `--accent`，且这个暖色也要是"莫兰迪化"的（砖红/陶土/黛玫，不是亮橙亮红）。

> 落地：当前主题库都按低饱和同色系方向处理。新增/调色时，宁可更灰更静，不要更艳。

## 维度二 · 布局：卡片 + 分层

不是"排得整齐"，是切卡片、叠层次，让重点信息和辅助信息"长得不一样"。

- 用 `.card` 把信息切块，靠 `--surface → --surface-2` 的纯色台阶 + 细边框 + 柔和阴影叠出层次（`--shadow-sm/--shadow`）。
- 平铺一大块 = 没呼吸感。留白要够（`--space-6/7` 级别的间距），让卡片之间透气。
- 重点卡片可微微抬升（hover `translateY` + 阴影），辅助信息收进次级层。

## 维度三 · 风格：匹配内容气质

不是"做得好看"，是"这内容看上去就该是这样"。按内容类型选气质：

| 内容类型 | 气质方向 | 具体做法 |
|---------|---------|---------|
| 金融 / 商务（我们最常用） | **corporate elegance + data-driven** | 克制配色、数据为主角、清爽表格/KPI、留白稳重。默认走这条。 |
| 科技 / 效率 | futuristic minimalism | 极简、冷色、细线、玻璃质感（毛玻璃 backdrop-blur，**不靠渐变**）。 |
| 人文 / 阅读 | editorial sketch | 杂志式排版、衬线标题、大字、留白叙事。 |
| 生活 / 消费 | organic modern + soft geometry | 柔和几何、更大圆角、温暖中性。 |

> 我们的看板/报告基本都属"金融商务"——默认就用 corporate elegance：默认 `gov-finance-blue`（政金蓝）主题 + 大 KPI + 清爽表格。需要更温和的内部说明再切 `sage` 或 `warm-sand`。

## 维度四 · 哲学：可亲近的高级感（approachable luxury）

高级但不端着。三个抓手：

1. **留白够**：宁可空，不要挤。
2. **细节克制**：柔和的 hover、恰到好处的圆角（`--radius`）、轻阴影；不堆特效。
3. **一两处人味**：一行暖文案、一个略带辨识度的标题字，让它有温度而不是冷冰冰的报表。

## 一段可直接用的"定调"提示词

> 配色用**莫兰迪低饱和 + 单色相**；布局用**卡片 + 分层留白**；风格按内容气质匹配
> （金融商务 → corporate elegance、数据为主角）；整体追求**可亲近的高级感**：留白够、
> 细节克制、留一两处人味。全程**纯色无渐变**，颜色只引用 token。

好设计不靠运气，靠精确的语言 + 审美。这一篇就是把审美变成可执行的语言。
